Portfolio of clients

This module allows generating a portfolio of clients and associating it with a seller.

After the customer portfolio is created, the customer file in the seller field takes the assigned one. A customer portfolio can not be deleted if you have assigned customers and and salesperson. it must first be removed from the customer file.

Customer file

Only the user who has the permission of sales manager can create and assign sellers and customers.
